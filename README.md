# contacts-app

# 📇 Contact Manager App (Flask)

Application web de gestion de contacts développée avec Flask et SQLite.

---

## 🚀 Description

Cette application permet de gérer une liste de contacts avec un système complet CRUD :

- Ajouter un contact
- Modifier un contact
- Supprimer un contact
- Rechercher un contact
- Afficher tous les contacts
- API JSON pour utilisation mobile

---

## 🧱 Technologies utilisées

- Python 🐍
- Flask (backend)
- SQLite (base de données)
- HTML / CSS
- Bootstrap 5 (interface responsive)
